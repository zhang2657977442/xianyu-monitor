+ 无需登陆，没有封号风险
+ 筛选条件强大，捡漏高效

## 设置凭证

只需3步即可设置凭证
1. 打开闲鱼官网 https://www.goofish.com 右键检查或F12打开开发者工具
2. 找到 **_m_h5_tk** 和 **_m_h5_tk_enc**
3. 填入保存即可

***因无需登陆，凭证有效时长2小时，登陆后再获取可延长时效***
